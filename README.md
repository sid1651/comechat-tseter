This repository contains my full implementation of the CometChat React UI Kit, generated using the UI Kit Builder.
The project includes setup instructions, environment configuration, screenshots, performance testing (Lighthouse), and a detailed analysis of issues & improvements.

üöÄ How to Run the Project Locally
1. Clone the repository
git clone https://github.com/sid1651/comechat-tseter.git
cd comechat-tseter

2. Install dependencies
npm install

3. Create a .env file in the project root
REACT_APP_COMETCHAT_APP_ID=your_app_id_here
REACT_APP_COMETCHAT_REGION=your_region_here
REACT_APP_COMETCHAT_AUTH_KEY=your_auth_key_here

4. Start the development server
npm start


The React UI Kit will start on http://localhost:3000
